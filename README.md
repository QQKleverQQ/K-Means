K-Means Clustering on Iris Dataset
English:

This project implements the K-Means clustering algorithm from scratch in Python using numpy and visualizes the clustering process with matplotlib. The algorithm is applied to the Iris dataset from scikit-learn.

Features:
Assign points to the nearest cluster center.
Recompute cluster centers as the mean of points in each cluster.
Check convergence based on the change in cluster centers.
Automatically run K-Means with 3 clusters.
Optional: visualize each iteration with points colored by cluster.

Usage:
python main.py
The script will display the iterative clustering process, showing how cluster centers move and points are assigned.

Dependencies:
Python 3.x
numpy
matplotlib
scikit-learn

Русский:

Этот проект реализует алгоритм кластеризации K-Means с нуля на Python с использованием numpy и визуализирует процесс кластеризации с помощью matplotlib. Алгоритм применён к датасету Iris из scikit-learn.

Особенности:
Сопоставление каждой точки ближайшему центру кластера.
Пересчёт центров кластеров как среднее значение точек в кластере.
Проверка сходимости на основе изменения центров кластеров.
Автоматический запуск K-Means с 3 кластерами.
Опционально: визуализация каждой итерации с разными цветами для каждого кластера.

Использование:
python main.py
Скрипт покажет процесс кластеризации на каждой итерации, как двигаются центры кластеров и как точки распределяются по кластерам.

Зависимости:
Python 3.x
numpy
matplotlib
scikit-learn
